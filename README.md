# Capstone_Wipro

1) After downloading, open up the zip keep it ready.
2) Database is already given, make the database in your system, probably workbench.
3) Create a workspace in Eclipse or VSC.
4) You can import the Eureka Server and Pizza Project as existing Maven into the created a workspace.
5) Run first Eureka Server using spring boot app option.
6) Change your db details in application properties file of Main folder i.e. pizzaportalultimate
7) Run it as Spring boot app
